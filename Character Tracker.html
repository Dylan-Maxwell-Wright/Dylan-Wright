Description

     Recently, I have begun playing Dungeons and Dragons with groups of friends. I had never truely known what the game entailed, and always assumed it was mainly about numbers and spreadsheets. Now, I have discovered that the game is full of all the best aspects of any game: it is very social, it involves critical thinking to solve problems set up by the game runner, teamwork is essential since no one person is the main character, and most importantly for me, it forces me to roll play as my character, which can be equally awkward and fearful as it can be fun and rewarding.

     With that in mind, I created this program as a fun project to accompany my D&D games. The Character Tracker is a console program which primarily deals with user input and output, and saving and loading data. This was the first project I built from scratch, and I learned a lot about the complexities of seemingly simple parts of C++. The program reacts adaptively to user input, allowing them to store an already made character, or make a brand new character, and easily modify their attributes, permenant statitics, and inventories. It is also prepared to handle errors in user input, and will not forcibly exit until the user asks.
